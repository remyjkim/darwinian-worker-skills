# semantic-memory

Durable runtime memory is semantic: observations are append-oriented JSONL,
and insights are authored Markdown. Both live in BeginningDB's canonical pool
and are placed into the selected Worker's Mind views by inode identity.

Cards declare the capability and seed persona or beliefs. They do not bundle
runtime memory entries. Use dedicated Mind tools for live observations,
insights, additional views, diagnostics, synchronization, and retirement.
